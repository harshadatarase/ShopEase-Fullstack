\# ShopEase - Full Stack E-Commerce Platform



A production-grade e-commerce platform built with Spring Boot and React.



\## Tech Stack

\### Backend

\- Spring Boot 3

\- PostgreSQL

\- Redis (Session Cart)

\- Elasticsearch (Search)

\- Razorpay (Payments)



\### Frontend

\- React + TypeScript

\- Vite

\- React Router



\## Features

\- Product listing with categories

\- Elasticsearch full-text search

\- Redis session-based cart

\- Razorpay payment integration

\- Admin panel (CRUD)

\- Role-based access control

\- Order management

\- User profiles



\## Setup

\### Backend

cd backend

mvn spring-boot:run



\### Frontend

cd frontend

npm install

npm run dev
